A polished, minimal, and responsive React app for a restaurant table booking flow with tasteful motion, spacing, and Apple-like blue accents.

## Tech Stack
- React + Vite
- React Router
- Tailwind CSS
- Framer Motion (micro-animations)
- LocalStorage (no backend)

## Run
```bash
npm install
npm run dev
```
Build:
```bash
npm run build
npm run preview
```

## Pages
- Home / Discover
- Restaurant Details
- Booking
- Confirmation
- My Bookings
- 404
